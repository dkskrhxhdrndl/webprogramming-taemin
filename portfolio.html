웹 게시판 시스템

설명: Python과 Flask로 구현한 웹 게시판 시스템. MariaDB를 연동하여 사용자/게시글/댓글 데이터를 관리하며, HTML/CSS를 통해 기본적인 UI를 제공합니다.

기능:

  • 회원 관리
  • 게시글 작성/조회/수정/삭제 (CRUD)
  • 댓글 작성 기능
  • 페이징, 검색 등 기본 UX

사용 기술: Python, Flask, MariaDB, Faker, HTML, CSS

역할: 전체 시스템 설계 및 데이터 생성 자동화 구현

성과: 10,000개 더미 데이터 삽입 성공 / GitHub 페이지에 설명 및 코드 업로드

Unity 미니게임

설명: Unity 2D 엔진을 활용한 타워 디펜스 스타일의 미니게임 구현. 유닛 배치, 웨이브 생성, 체력 및 점수 시스템을 포함.

기능:

  • 2D 유닛 생성 및 배치
  • 적 웨이브 출현 및 전투 시스템
  • HP/점수/게임오버 UI 구현
  • 기본 게임 룰 및 밸런싱

사용 기술: Unity 6, C#, Photoshop (UI 디자인 일부)

역할: 전체 게임 시스템 기획 및 구현

성과: Unity 학습 기반으로 기초 게임 시스템 직접 제작
